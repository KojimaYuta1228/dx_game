●収録内容

魔法陣が表示されるアニメーションをいくつかの角度とカラーバリーションを用意したエフェクトアニメ素材です

RPGなどの戦闘シーンやイベントシーンでの魔法発動効果での使用を想定したものになっています

収録している画像データは、アニメパターンをスプライトシート状に並べたPNG画像となっています

アニメーションの表示速度は15fpsで作成されています

ファイルサイズの最適化はしておりませんので、必要に応じて下記の「注意事項」で紹介しているアプリなどでファイルサイズを小さくすることをお奨めします

「オーラ_地面のみ」フォルダ

地面表示の魔法陣に重ねて表示することができるオーラエフェクトアニメ素材になります

ただし、「フェードインアウト　地面」にはフェードアウトしてしまうアニメなので、こちらでの使用には向きません

分割パーツについて

オーラエフェクトアニメ素材は、こちらの画像のようにキャラクターなどを真ん中に挟んで表示できるようオーラの手前と奥を別画像にしたものも用意してあります

こちらの画像の場合、一番下に「魔法陣エフェクト」、その上に「オーラエフェクト奥」、その上に「キャラクター」、一番上に「オーラエフェクト手前」の順番で表示しています
